Imersão Prática de Posicionamento

Objetivo: Construir um portfólio completo, do zero. A cada atividade, vamos adicionar um novo pedaço de código HTML e usar CSS para posicioná-lo, estruturando a página passo a passo.

O que é Flexbox? Entendendo a Estrutura

Flexbox é um modelo de layout para organizar elementos em uma linha ou em uma coluna. Para usá-lo, precisamos de um Container (o elemento pai) e de Itens (os elementos filhos).

O conceito mais importante são os eixos:

  • Eixo Principal (main axis ): É a direção principal do alinhamento (por padrão, horizontal). Usamos justify-content para alinhar os itens neste eixo.
  • Eixo Cruzado (cross axis): É o eixo perpendicular (por padrão, vertical). Usamos align-items para alinhar os itens neste eixo.

Atividade 1: A Estrutura Básica do Site

Vamos começar criando nossos arquivos e a estrutura HTML e CSS mais básica.

Exercício 1: Criando a Base

1. Crie um arquivo `portfolio.html` com o código fundamental de um documento HTML.

<!DOCTYPE html> <html lang="pt-br"> <head> <meta charset="UTF-8"> <title>Meu Portfólio</title> <link rel="stylesheet" href="style.css"> </head> <body> <!-- Nosso site será construído aqui dentro --> </body> </html>

2. Crie um arquivo `style.css` e adicione o CSS para zerar as margens e definir uma cor de fundo.

body { font-family: sans-serif; margin: 0; background-color: #f4f6f7; }
Sua página estará em branco. Isso é normal! Vamos começar a adicionar conteúdo.

Atividade 2: Construindo e Alinhando o Cabeçalho

Vamos adicionar o `header` e usar Flexbox para alinhar o título e o menu.

Exercício 2: Posicionando o Menu

1. Adicione o HTML do header dentro da tag `` do seu `portfolio.html`.

<header> <h1>Meu Nome</h1> <nav> <a href="#sobre">Sobre Mim</a> <a href="#projetos">Projetos</a> </nav> </header>

2. Adicione o CSS do header ao seu `style.css` para posicionar os itens.

header { background-color: #2c3e50; color: white; padding: 20px 40px; display: flex; justify-content: space-between; align-items: center; } nav a { color: white; margin: 0 15px; text-decoration: none; font-size: 18px; }

Explore: Altere o valor de justify-content para center e flex-start. Qual você acha que fica melhor?

Printe o resultado do seu cabeçalho alinhado.

Atividade 3: Dando Vida ao Menu com `hover`

Um bom site responde ao usuário. Vamos adicionar um efeito simples de `hover` nos links.

Exercício 3: O Efeito `hover`

Adicione este código ao final do seu `style.css`.

nav a:hover { color: #1abc9c; }
A pseudo-classe :hover aplica o estilo apenas quando o mouse está sobre o elemento. Teste mudar a cor para outra de sua preferência!
Passe o mouse sobre um link, printe o efeito e adicione ao seu documento.

Atividade 4: Construindo a Seção de Apresentação

Esta é a primeira seção de conteúdo do site. Vamos criá-la e estilizá-la.

Exercício 4: Centralizando Conteúdo

1. Adicione o HTML da apresentação no `portfolio.html`, logo após o ``.

<main> <section class="apresentacao"> <h2>Olá, eu sou [Seu Nome]</h2> <p>Desenvolvedor(a) em formação, apaixonado(a) por criar soluções com tecnologia.</p> </section> </main>

2. Adicione o CSS da apresentação ao `style.css`.

.apresentacao { text-align: center; padding: 80px 20px; }
text-align: center; alinha o conteúdo de texto. O padding cria o espaçamento interno.
Printe o resultado da sua nova seção de apresentação.

Atividade 5: Construindo um Layout de Duas Colunas

Agora, um desafio comum: criar uma seção com texto de um lado e imagem do outro.

Exercício 5: Dividindo o Espaço com `flex`

1. Adicione o HTML da seção "Sobre" no `portfolio.html`, dentro da tag `

`.

<section id="sobre" class="sobre"> <div class="sobre-texto"> <h2>Sobre Mim</h2> <p>Escreva um texto real sobre você. Fale sobre seus interesses, o que está aprendendo e seus objetivos.</p> </div> <div class="sobre-imagem"> <img src="https://via.placeholder.com/400" alt="Imagem Sobre Mim"> </div> </section>

2. Adicione o CSS para transformar a seção em um container flex.

.sobre { background-color: white; padding: 60px 5%; display: flex; align-items: center; gap: 40px; } .sobre-texto { flex: 1; } .sobre-imagem { flex: 1; } .sobre-imagem img { width: 100%; border-radius: 10px; }
Printe o resultado da sua seção de duas colunas.

Atividade 6: Explorando o Alinhamento Vertical

A propriedade `align-items` controla como os itens se alinham no eixo vertical. Vamos testá-la.

Exercício 6: Testando `align-items`

Altere o valor de align-items na classe .sobre para flex-start e depois para flex-end.

Para ver o efeito claramente, adicione mais texto no parágrafo da seção "Sobre Mim" no seu HTML. Você verá a imagem se alinhando ao topo, ao centro ou à base do texto.
Escolha o alinhamento vertical que preferir e printe o resultado.

Atividade 7: Construindo os Cards de Projetos

Vamos adicionar a seção de projetos. Por padrão, os cards ficarão empilhados. No próximo exercício, vamos alinhá-los.

Exercício 7: Adicionando os Cards

1. Adicione o HTML dos projetos no `portfolio.html`, dentro da tag `

`.

<section id="projetos" class="projetos"> <h2>Meus Projetos</h2> <div class="cards-container"> <div class="card"><h3>Projeto 1</h3><p>Descrição do projeto.</p></div> <div class="card"><h3>Projeto 2</h3><p>Descrição do projeto.</p></div> <div class="card"><h3>Projeto 3</h3><p>Descrição do projeto.</p></div> </div> </section>

2. Adicione o CSS para estilizar a seção e os cards.

.projetos { padding: 60px 5%; text-align: center; } .card { background-color: white; padding: 20px; border-radius: 8px; box-shadow: 0 4px 15px rgba(0,0,0,0.1 ); margin-bottom: 20px; /* Apenas para separar enquanto estão empilhados */ }
Printe os cards ainda empilhados. Este é o nosso "antes".

Atividade 8: Alinhando os Cards com Flexbox

Agora vamos usar a mágica do Flexbox para alinhar os cards que acabamos de criar.

Exercício 8: Alinhando e Distribuindo

Adicione este código ao seu `style.css`.

.cards-container { padding: 20px 0; display: flex; justify-content: center; gap: 30px; flex-wrap: wrap; /* Permite que os cards quebrem a linha em telas menores */ } /* Vamos ajustar o card para o modo flex */ .card { flex: 1; /* Faz cada card tentar ocupar o mesmo espaço */ max-width: 350px; /* Impede que fiquem largos demais */ margin-bottom: 0; /* Não precisamos mais da margem de separação */ }
Explore: Diminua a largura da janela do seu navegador. Graças ao flex-wrap: wrap;, os cards que não cabem na linha "caem" para a linha de baixo. Isso é a base da responsividade!
Printe sua seção de projetos com os cards lindamente alinhados.

Atividade 9: Adicionando Efeitos de `hover` nos Cards

Vamos dar um feedback visual para o usuário, fazendo os cards "saltarem" ao passar o mouse.

Exercício 9: Efeito de Elevação

Adicione este código ao seu `style.css`.

.card { /* ... outras propriedades que já estavam lá ... */ transition: transform 0.3s; /* Define a animação suave */ } .card:hover { transform: translateY(-10px); /* Move o card 10px para cima */ }
A propriedade transition é a chave para a suavidade. Tente removê-la e veja como o efeito fica "duro" e instantâneo.
Passe o mouse sobre um card, printe o efeito e adicione ao seu documento.

Atividade 10: Finalizando com o Rodapé

Todo bom site precisa de um rodapé. Vamos criar o nosso e finalizar o projeto.

Exercício 10: Construindo o `footer`

1. Adicione o HTML do rodapé no `portfolio.html`, após o fechamento da tag ``.

<footer> <p>© 2025 - Portfólio de [Seu Nome]</p> </footer>

2. Adicione o CSS final para estilizar o rodapé.

footer { background-color: #2c3e50; color: white; text-align: center; padding: 30px; margin-top: 60px; }
Parabéns! Você construiu um portfólio completo do zero. Printe a tela final do seu projeto.
